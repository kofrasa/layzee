# Changelog

## 1.0.0 / 2020-05-23

- Convert to typescript
- Unexport `isIterable`
- Remove `max` and `min`
- Rename `one` to `first`
- Add docs

## 0.0.0 / 2018-01-31

- Chainable methods:
  - `map`, `filter`, `take`, `takeWhile`, `drop`, `dropWhile`,
  - `sample`, `uniq`, `sort`, `sortBy`, `reverse`, `transform`
- Sequence methods:
  - `each`, `reduce`, `size`, `max`, `min`, `groupBy`, `partition`, `one`, `value`
- Helper methods:
  - `range`, `isIterable`
